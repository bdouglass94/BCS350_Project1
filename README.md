# BCS350_Project1
# GameZify.com – Web Design Document

## Project Overview
GameZify is a fictional online store for games on PlayStation, Xbox, and Nintendo Switch. Users can browse, buy, and trade-in games through an interactive website. 

--------------------------------------------------------------------------------------

## Target Audience
Our target audience is gaming enthusiasts and casual gamers looking for a place to purchase all of there gaming needs.

--------------------------------------------------------------------------------------

## Content Strategy
Pages: Home, Shop, About, Contact, Cart, Trade-In 
Tone: Friendly and playful 
Key Action buttons: “Shop Now”, “Trade Now”, “Send Message” 
Featured platforms and games are highlighted on homepage 

--------------------------------------------------------------------------------------

## Visual Design
The visual design is to provide a nostlagic feeling gaming website. It utilizes neon colors and pixelated fonts and imagery to create that 8-bit gaming design.

The website has clickable links and various pages with text and images imbedded into the design. There are buttons to navigate to different consoles and games Shapeand buttons to add these items to cart for checkout.

PART 2: DESIGN (Answer ALL questions, add your own if necessary) 

1. What overall style will the website have (minimalist, corporate, creative, etc.)? 
The website will have a creative, fun gaming design to bring in gamers of all ages. 

 
2. What color scheme will you use and why? 
It will use neon style green and black color background giving it a cool gaming asthetic throughout the website. 

 
3. What fonts will you use for headings and body text? 
The website uses Press Start 2p font which is a pixelated style font giving the site a nostalgic gaming feel. 

 
4. How will spacing and layout improve readability? 
Spacing is being managed by adding padding, margins, and section separators to keep the page from feeling cluttered. Different contents of the pages are separated by different sections, so content is easily readable. 
 

6. What visual hierarchy will guide the user’s attention? 
The visual hierarchy begins with the hero section containing the Shop Now button. Below is a console platform category to select the genre of games you want to browse. Bright button colors are used to highlight important action buttons on the website. Featured games are below that which can be added to cart and finallly a trade in game sction at the bottom along with social media links in the footer. 

 
7. How will consistency be maintained across pages? 
Consistency will be maintained by using the same navigation bar, fonts, color scheme, and layout structure across all pages. Each page will include the same header and footer, so users always know how to navigate the site. 


8. What icons will you use? 
The site will use game related icons including console images, game images and a shopping cart icon for your final cart before checkout. 

 
9. What emotions should the design evoke in users? 
The site is suppposed to invoke feelings of excitement, nostalgia for users purchasing older games, and a sense of community engagement. 

 
10. How will accessibility be considered (contrast, font size, etc.)? 
Accessibility will be considered by using high contrast colors, such as bright neon green text against a dark background. The layout is also responsive, so the site works on different screen sizes, and images include alt text to support screen readers. 

 
11. What inspiration sources influenced your design? 
I always like the old 8-bit game style from back in the day. It always felt like to heart of original game design for me so that is why I went with this style for my gaming site even though it isint a retro game website. 

 
12. Describe the layout of the homepage and build the wireframe. 
The homepage starts with the navigation header which containts the links to the main pages of the site. Below the header is a hero banner that introduces the website and includes a call-to-action button encouraging users to shop. The next section displays platform categories, allowing users to browse for game based on the consoles catgegory. Following this a featured games section where individual games are displayed with images, prices, and add-to-cart buttons. A trade-in banner promotes the trade-in feature of the site. Finally, the page ends with a footer containing copyright and social media links.

PART 3: INTERACTIVITY (Answer ALL questions, add your own if necessary) 

1. What interactive elements will your site include (forms, buttons, menus, etc.)? 
The website will use interactive elements including navigation links, buttons, add to cart buttons, and trade in and contact forms allowing users to submit requests on the website. 


2. How will users navigate the website? 
Users will navigate the site using the navigation header that is made available throughout the websites pages. 

 
3. Will there be form validation or user feedback? Explain. 
Yes, there will be form validation that allows users to enter their name, email, and a fill a message text box before submitting the form.  

 
4. What JavaScript functionality will you implement? 
JavaScript will be used to add functionality such as adding games to the shopping cart, displaying alerts or feedback when items are added, and handling form validation for the contact page. 

 
5. How does interactivity improve user experience? 
Interactivity improves user experience by allowing users to actively engage with the website. Features such as clickable buttons, add-to-cart functionality, and contact forms make the site more dynamic and easier to use compared to a static page.
